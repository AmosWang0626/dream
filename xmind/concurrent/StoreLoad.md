# Store Load

- store：处理器将数据写入写缓冲器
- load：从高速缓存里读数据

## 内存重排序四种可能

1. LoadLoad
   一个处理器先执行L1读操作，再执行L2读操作；但是另一个处理器看到的是先L2再L1。
2. StoreStore
   一个处理器先执行W1写操作，再执行W2写操作；但是另一个处理器看到的是先W2再W1。
3. LoadStore
   一个处理器先执行L1写操作，再执行W2写操作；但是另一个处理器看到的是先W2再L1。
4. StoreLoad
   一个处理器先执行W1写操作，再执行L2写操作；但是另一个处理器看到的是先L2再W1。
